# FormPass

就活マイページの氏名、住所、連絡先などを1クリックで入力するChrome拡張です。

## できること

- プロフィールをChromeローカルストレージに保存
- 現在ページのinput、textarea、select、radio、checkboxへ自動入力
- 日本語ラベル、placeholder、name、id、近接テキストから項目を推定
- Axol系の氏名2分割、生年月日select、郵便番号/電話番号分割入力に対応
- 休暇中の連絡先住所を保存・自動入力
- ゼミ・研究室、クラブ・サークルを保存・自動入力
- ポップアップで自動入力されるプロフィール内容を確認
- プロフィールのJSONエクスポート、インポート
- パスワードなどの認証情報は保存・自動入力しない

電話番号と郵便番号は、設定画面ではハイフンあり・なしのどちらでも保存できます。
Axolのような分割入力欄では、拡張側で自動的に `000 / 0000 / 0000`、`000 / 0000` のように分けて入力します。
郵便番号検索ボタンがあるフォームでは、郵便番号入力後に検索ボタンを押し、反映後に保存済みの町域・番地や建物名を再入力します。

## 読み込み方法

1. Chromeで `chrome://extensions` を開く
2. 右上の「デベロッパー モード」をON
3. 「パッケージ化されていない拡張機能を読み込む」を押す
4. `/Users/sakai/Downloads/formpass` を選択
5. 拡張機能の「設定」からプロフィールを保存
6. マイページのフォームを開いて「このページに入力」を押す

`test-form.html` をChromeで開くと、手元で入力推定を試せます。

## 方針

送信ボタンの自動クリックは実装していません。入力後にユーザーが確認して送信する前提です。
パスワードなどの認証情報はChrome標準のパスワードマネージャーや専用ツールに任せ、この拡張では扱いません。
